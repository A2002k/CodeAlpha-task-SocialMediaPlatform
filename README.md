## 📖 About

SocialSphere is a modern social networking application where users can:

- Create and share posts
- Like and comment on posts
- Follow other users
- Send private messages
- Create Instagram-style stories
- Tag friends in posts
- Receive real-time notifications
- Repost content
- Edit images before publishing

The goal of this project is to demonstrate full-stack development skills using modern web technologies.
